# Digital Literacy Project

This repository contains the completed tasks for my **Digital Literacy Assignment**. The project demonstrates fundamental skills in web development (HTML/CSS), digital documentation, and online information management.

## 🚀 Project Overview
The goal of this assignment was to apply digital literacy principles through practical technical tasks. It covers the creation of structured web content, styling, and organized digital workflows.

## 📂 Repository Structure & Task Descriptions

### Task 1: Basic Web Structure (HTML)
* **File/Folder:** `task1/` or `index.html`
* **Description:** This task focused on creating a well-structured HTML document. It includes the use of semantic tags (header, footer, section), lists, and hyperlinks to ensure web accessibility and proper document hierarchy.

### Task 2: Visual Styling (CSS)
* **File/Folder:** `task2/` or `style.css`
* **Description:** In this task, I applied CSS to the HTML structure. Key concepts demonstrated include the Box Model, typography management, color schemes, and basic layout techniques (like Flexbox or Grid) to enhance user experience.

### Task 3: Information Organization & Documentation
* **File/Folder:** `docs/` or `Task3_Report.pdf`
* **Description:** This part of the project involved synthesizing information and presenting it in a clear, digital format. It reflects the ability to curate content, cite sources correctly, and maintain a professional document layout.

### Task 4: Digital Workflow & Version Control
* **Description:** This repository itself serves as evidence of digital literacy by utilizing Git for version control. It shows an understanding of how to manage project history, branch management, and collaborative coding environments.

## 🛠️ Tools Used
- **HTML5 & CSS3:** For building and styling the web components.
- **Git & GitHub:** For version control and project hosting.
- **VS Code:** As the primary code editor.

## 🏁 Conclusion
Through this assignment, I have strengthened my ability to navigate digital platforms, create structured online content, and manage technical projects efficiently.
